Lead Story

Built a Fully Serverless Portfolio CMS on Firebase

Gaurav Sharma engineers a 100% serverless, glassmorphism-styled project management platform leveraging React, Vite, and Firebase Cloud Functions.
JAIPUR, RAJASTHAN Technology Desk
System Architecture
Platform Project Management Dashboard
Arch 100% Serverless
Frontend React + Vite + Tailwind CSS
Backend Firebase Cloud Functions
Database Firestore + Firebase Storage

The core problem was eliminating hardcoded HTML and legacy PHP hosting costs for portfolio management. A 100% serverless architecture was engineered using Firebase Cloud Functions and Firestore to create a dynamic, secure Admin Dashboard.

Implementation Detail: The 15-Minute Rule

To safely allow public write access to the live dashboard demo without database bloat, an automated 15-minute data-wipe Cloud Function was developed. Any data created by a guest user is systematically purged, ensuring the dashboard permanently resets itself.

INDUSTRY DESK

The native HTML5 Canvas API paired with Vanilla JS was selected to ensure maximum performance and zero dependency overhead. A custom unlimited undo/redo stack was engineered to track canvas state changes without memory leaks, operating alongside a unified touch/mouse event handler.

Needed a reliable way to calculate risk-based lot sizing across volatile asset classes. Integrated ExchangeRate-API for real-time currency conversion rather than static tables, and developed custom instrument-aware pip logic to automatically adjust decimal precision based on the asset type (standard pair vs JPY cross).
