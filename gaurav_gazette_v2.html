Jaipur Developer Ships Full-Featured Browser Painting App

GS Paint brings watercolour simulation, airbrush tools, and unlimited undo to any browser — with complete touch support for tablet artists.
JAIPUR, RAJASTHAN Technology Correspondent
GS Paint — Canvas API

GS Paint demonstrates advanced browser-native graphics programming using the Canvas API. The application supports pencil, marker, watercolour, and airbrush tools with adjustable opacity and brush size, alongside shape primitives — line, rectangle, circle, and triangle.

An eraser tool and fill bucket round out the editing suite. An unlimited undo/redo stack handles complex multi-step artwork without data loss. Touch events are fully supported, making the application functional on mobile and tablet devices without a separate codebase.

STACK: Canvas API · Vanilla JS · Touch Events

Forex Calculator Suite: Live P&L and Lot Sizing for Active Traders
FINTECH DESK

Powered by ExchangeRate-API v6. Supports all major FX pairs, Gold, and Silver with instrument-aware pip logic. Risk-based lot sizing displays exact leverage required per position.

The P&L Calculator converts profit and loss to the user's account currency in real time. The Lot Size Calculator derives optimal position size from a user-defined risk percentage. Both tools handle Gold and Silver with distinct pip calculations separate from standard FX logic.

STACK: ExchangeRate-API v6 · Vanilla JS
