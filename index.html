Serverless Admin Dashboard Revolutionises Dynamic Portfolio Management

Gaurav Sharma engineers a 100% serverless, glassmorphism-styled project management platform leveraging React, Vite, and Firebase Cloud Functions.
JAIPUR, RAJASTHAN Technology Desk
System Architecture
Platform Project Management Dashboard
Arch 100% Serverless
Frontend React + Vite + Tailwind CSS
Backend Firebase Cloud Functions
Database Firestore + Firebase Storage

Designed to be a highly reusable, beautifully styled platform, this serverless application eliminates hardcoded HTML. Instead, it provides a secure, authenticated Admin Dashboard where professionals can dynamically add, edit, and categorize their work in real-time. The public view features stunning glassmorphism design tokens and micro-animations.

Live Dashboard Access
Live URL gs-admin-project-managment.netlify.app
Anyone can create an account and test all the features by themselves.

⏱️ 5-Minute Auto-Delete Rule: Anything you create inside the dashboard is automatically wiped after 5 minutes via a Firebase Cloud Function — so feel free to add, edit, and delete projects without holding back. The dashboard always resets itself.

Technical Highlights & Security

The system successfully migrated from a legacy PHP backend to Firebase, eliminating hosting costs. It features complete CRUD capabilities for projects and custom categories, secure image uploading via Firebase Storage, and strict Firestore security rules with CORS configuration ensuring only authenticated admins can manipulate data.

STACK: React · Vite · Firebase Auth · Firestore · Cloud Functions
